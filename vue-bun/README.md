# ORPC Playground

This is a playground for [oRPC](https://orpc.unnoq.com) and [Vue](https://vuejs.org).

## Getting Started

First, run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:2999](http://localhost:2999) with your browser to see the result.
